Welcome to the File Explorer App!

This application allows you to navigate through a directory structure and view various types of files directly in your browser. Here's what you can do:

  1. Browse Directories: Navigate through folders and subfolders to find the files you need.
  2. View Files: Click on a file to view its content. Supported file types include images (PNG, JPG, GIF, SVG), videos (MP4, AVI), and various text-based files (TXT, PY, JS, HTML, CSS, CPP, MD).
  3. Download Files: Download any file by clicking the download button provided.
